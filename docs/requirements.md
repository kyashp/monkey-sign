# User Requirements

## About
MonkeySign is a Monkeytype inspired speed signing game focused on Singapore Sign Language (SGSL). The web application is for users that are interested to gain fluency in signing SGSL.

## Functional Requirements

### Basic
1. User starts a signing session when spacebar clicked
2. User sees a countdown from 3 to start signing
2. User sees a timer in the bottom left corner, counting down from 1 min (mm:ss)
3. User sees their own webcam footage
4. User sees the sign and the interpretation on the web app
5. User's webcam footage has border green when correct, border grey when undetected and border red when wrong sign
6. User's session ends when timer reaches 0
7. User sees their signing statistics, SPM
8. A User can begin new session by pressing Tab then Enter

## Non-Functional Requirements
1. Latency between user signing and detection should be within 100ms
2. Data Privacy concerns addressed through a pop up/ page
